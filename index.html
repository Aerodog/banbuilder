BOO Censorship!

I hate censorship as much as anyone, but as a web application developer, there are times when a banned-word list is necessary, especially if the application or site is geared towards younger users or corporate environments.

In the repo, over 800 naughty words are available in the following formats:

  • SQL
  • PHP Array
  • CSV
  • LaTex
  • YML
  • XML

How to Get It?

Just click on the tar.gz or .zip icons above to download the files. Or if you prefer, you can clone the repo: 

$ git clone git@github.com:snipe/banbuilder.git

Warning, Not For All Audiences, Blah Blah Blah...

There are over 800 words in the database so far. It should probably go without saying that this content is NOT SAFE FOR WORK and is NOT appropriate for children or anyone who is easily offended by words.

And not to overstate the obvious, but no banned-word list is going to be flawless. A G-rated list will block out the word "screw", but there are certainly legitimate uses for the word "screw". Also, the word "dick" might be banned, but many men named Richard are called "Dick" as their nickname. You may need to edit your list to suit your needs, removing some words from the banned-word list that you feel are too restrictive.

About This Project

On the forums I run, I tend to be fairly lenient on what my users can get away with, blocking only the most offensive and racist terms. This project started after I was forced to create and review yet another banned word list, this time for a client project targeted towards young adults. The banlist needed to be locked down pretty tight, which meant that my standard list of banned words wouldn't even come close to fitting the bill.

As I was putting this list together, I reached out to my filthy-mouthed friends to gather words for the ban list. It occurred to me that collectively, we could accomplish a lot more.
